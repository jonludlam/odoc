Deps

The following libraries are dependencies of odoc.

  • Stdlib
  • Fpath
  • Cmdliner
  • Astring
  • Result
  • Yojson
  • Tyxml

Stdlib

Stdlib The standard library of OCaml.

Fpath

Fpath provides a library for handling file system paths.

Cmdliner

Cmdliner provides a library for handling command-line arguments.

Astring

Astring provides a replacement for the standard OCaml String module.

Result

Result is the Result Monad module.

Yojson

Yojson is a JSON library.

Tyxml

Tyxml is the library used to generate HTML pages.
